JavaScript Notes

JavaScript is a high-level programming language used to create interactive and dynamic web pages. 📌 Core Concepts: • Variables: let, const, var • Data Types: String, Number, Boolean, Object, Array • Operators • Conditions (if / else) • Loops • Functions • DOM Manipulation 📌 Example: let age = 20; if(age >= 18){ console.log("Adult"); } 📌 Importance: Used in almost every modern website for interactivity.

Algebra Basics

Algebra is a branch of mathematics that uses symbols and letters to solve equations. 📌 Core Topics: • Variables (x, y, z) • Linear Equations • Quadratic Equations • Factorization 📌 Example: 2x + 4 = 10 2x = 6 x = 3 📌 Importance: Helps build logical thinking and problem-solving skills.

Business Studies

Business Studies explains how companies operate and manage money. 📌 Core Areas: • Types of Business • Marketing (4Ps) • Management • Finance • Entrepreneurship 📌 Example: Product, Price, Place, Promotion 📌 Importance: Helps understand real-world business systems.

Digital Marketing

Digital marketing is promoting products using the internet and social media. 📌 Types: • SEO (Search Engine Optimization) • Social Media Marketing • Email Marketing • Paid Ads 📌 Example: Instagram ads used by companies to promote products. 📌 Importance: Helps businesses reach a global audience quickly.
